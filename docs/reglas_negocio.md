# Reglas de Negocio (UX/UI)

Para garantizar la adopción de **FerreStock Pro** por parte del personal de la ferretería (usuarios con potencial baja alfabetización digital):

1. **Interfaz 100% Táctil/Icónica:** Las operaciones core de la ferretería (ubicar productos, añadir/retirar stock, consultar existencias) deben realizarse únicamente interactuando con elementos visuales grandes y representativos en pantalla (toques/clics).
2. **Cero Dependencia de Teclado:** Evitar al máximo los campos de entrada de texto ("inputs" tipados). El uso del teclado será exclusivamente para casos administrativos o la primera alta del negocio, nunca para la operativa diaria en mostrador.
3. **Ausencia de Códigos de Barras:** El sistema no asume ni tolera procesos manuales dependientes de lectores y etiquetas con códigos. La localización y control es estrictamente espacial y asociativa.
4. **Resiliencia Operativa:** La UI debe ser robusta, minimizar pasos de fricción y utilizar colores amigables (semáforos) previniendo errores por interacciones erróneas (espacios holgados y padding alto para interacciones dactilares táctiles).
