# Arquitectura y Stack Tecnológico

El stack del proyecto **FerreStock Pro** está diseñado para ser ligero, capaz de instalarse localmente (prioridad para ordenadores modestos o tablets en mostradores de ferreterías) y con un alto enfoque visual y responsivo.

- **Frontend / UI:** React (para el manejo del DOM y estados reactivos por interacción).
- **Estilos:** Tailwind CSS (optimizado para construir interfaces, grillas y componentes táctiles de manera rápida, flexible y consistente).
- **Base de Datos / Backend Local:** SQLite (para almacenamiento local con requerimientos mínimos, sin necesitar servidores pesados o internet constante, ideal para comercios de barrio).
